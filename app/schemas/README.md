# Room Schema Export

This directory contains JSON schema export files generated by Room.
They are used to track database migration history.

Room will auto-generate a file here (e.g., `1.json`) when you build the project,
if `exportSchema = true` in `AppDatabase`.

**To enable schema export**, add to `app/build.gradle.kts`:

```kotlin
ksp {
    arg("room.schemaLocation", "$projectDir/schemas")
}
```

Or in the `android` block:
```kotlin
defaultConfig {
    javaCompileOptions {
        annotationProcessorOptions {
            arguments += mapOf("room.schemaLocation" to "$projectDir/schemas")
        }
    }
}
```

The generated schema JSON will look like:
```json
{
  "formatVersion": 1,
  "database": {
    "version": 1,
    "identityHash": "...",
    "entities": [...]
  }
}
```
